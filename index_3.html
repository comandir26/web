Этот документ собрал вместе три очень важных и наиболее часто используемых Python-библиотек для работы с данными. Эти три модуля необходимы для работы с данными, числами и визуализацией. Большую роль играют в изучении искусственного интеллекта.

NumPy — это открытая бесплатная Python-библиотека для работы с многомерными массивами. NumPy чаще всего используют в анализе данных и обучении нейронных сетей — в каждой из этих областей нужно проводить много вычислений с такими матрицами.

Pandas - главная библиотека в Python для работы с данными. Её активно используют аналитики данных и дата-сайентисты. Она быстрая и мощная: в ней можно работать с таблицами, в которых миллионы строк.

Matplotlib — популярная Python-библиотека для визуализации данных. Она используется для создания любых видов графиков: линейных, круговых диаграмм, построчных гистограмм и других — в зависимости от задач.
